# Aufgabe — Gestor de Incidencias

**Aufgabe** es un gestor de incidencias pensado para **hospitales, geriátricos, escuelas... etc.**  
Su objetivo es facilitar la comunicación, el registro y el seguimiento de incidencias en entornos donde la organización y la respuesta rápida son fundamentales.


## Características principales

- Registro y gestión de incidencias en tiempo real.  
- Clasificación por tipo, prioridad y estado.  
- Panel de administración con control de usuarios.  
- Interfaz web moderna y adaptable.  
- Integración con bases de datos para la persistencia de información.  


## Tecnologías utilizadas

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Base de datos:** MySQL  
- **Entorno de desarrollo:** Visual Studio Code  
- **Control de versiones:** Git y GitHub
